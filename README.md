# AI Text Summarizer

A full-stack AI-powered text summarization web application built using Flask and React.

## 🚀 Features

- Upload PDF documents
- Extract text automatically
- Generate AI-based summaries using Hugging Face inference API
- Download summary as PDF
- Clean and simple UI

---

## 🛠 Tech Stack

### Frontend
- React
- CSS

### Backend
- Flask
- Hugging Face Inference API
- PyMuPDF (PDF text extraction)
- ReportLab (PDF generation)

---

## 🏗 Architecture

Frontend (React) → API Request → Flask Backend → Hugging Face Model → Summary → Return to Frontend
